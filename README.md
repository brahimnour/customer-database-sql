# customer-database-sql
Modélisation et analyse d'une base de données clients en SQL
#  Base de données clients & commandes

## Description
Modélisation et analyse d'une base de données relationnelle couvrant 
clients, produits et commandes. Les requêtes SQL extraient des insights 
métier pour aider à la prise de décision.

## Compétences utilisées
- Modélisation relationnelle (MCD / MLD)
- SQL : jointures, agrégations, sous-requêtes, filtres
- Analyse métier orientée prise de décision

## Modèle de données
| Table | Description |
|---|---|
| `Clients` | Informations clients |
| `Produits` | Catalogue produits et prix |
| `Commandes` | Commandes passées par les clients |
| `LignesCommande` | Détail des produits par commande |

## Analyses réalisées
- Chiffre d'affaires total
- Top 5 produits les plus rentables
- Clients les plus actifs
- Ventes par catégorie de produit
- Évolution mensuelle des ventes
- Panier moyen par client
